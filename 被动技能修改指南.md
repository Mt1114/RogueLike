## 🔧 修改示例

### 示例1：增强生命强化效果
```python
# 将生命强化从 50/100/200 改为 100/200/400
effects={'max_health': 100},  # 1级：100点生命值
effects={'max_health': 200},  # 2级：200点生命值
effects={'max_health': 400},  # 3级：400点生命值
```

### 示例2：增强速度提升效果
```python
# 将速度提升从 10%/20%/30% 改为 20%/40%/60%
effects={'speed': 0.2},  # 1级：20%速度提升
effects={'speed': 0.4},  # 2级：40%速度提升
effects={'speed': 0.6},  # 3级：60%速度提升
```

### 示例3：增强攻击力效果
```python
# 将攻击力提升从 10%/20%/30% 改为 25%/50%/75%
effects={'attack_power': 0.25},  # 1级：25%攻击力提升
effects={'attack_power': 0.5},   # 2级：50%攻击力提升
effects={'attack_power': 0.75},  # 3级：75%攻击力提升
```

## ⚠️ 注意事项

1. **百分比属性**：`speed`、`luck`、`attack_power`、`defense` 使用百分比计算
   - 0.1 = 10%
   - 0.5 = 50%
   - 1.0 = 100%

2. **固定数值属性**：`max_health`、`health_regen`、`pickup_range` 使用固定数值
   - 直接修改数值即可

3. **描述文字**：记得同时修改 `description` 中的描述文字，保持一致性

4. **测试**：修改后记得测试游戏，确保效果符合预期

## 🎮 修改后的效果

修改完成后，玩家选择相应的被动技能时，会立即获得修改后的属性加成，并且会实时反映在游戏中！
